Concert Slideshows Search Help

Search for slides by entering an artist in the search text and/or selecting a venue from the drop-down list followed by clicking the search button (🔍). An exclamation mark (!) as the first character in the search text indicates negation, i.e. matches all artists NOT containing the search text. Searches are NOT case-sensitive and can include whitespace to match multiple words (in order) within the artists' names.

For example, to search for slides from Linkin Park, type "link park" in the search text and click the search button. To search for slides from Moody Center that aren't Linkin Park, type "!link park" in the search text, select Moody Center from the drop-down list, and click the search button.
